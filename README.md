# ☘cloverx-starter

## Usage
安装依赖，只能使用`npm`
```shell
npm install
```
启动
```shell
npm run dev
```


## 目录结构
```shell
.
├── config  // 配置文件目录
│   ├── common.js
│   ├── development.js
│   ├── local.js
│   └── production.js
├── controller // 控制器目录
│   ├── client.js
│   └── home.js
├── extend
├── index.js
├── model   // 业务逻辑 和 数据逻辑
│   ├── client
│   │   └── version.js
│   └── hello.js
└── service
```
